# Lendwire File uploader

## Description

A basic app that upload files to s3



## Installation

1. Clone the repo:
   ```sh
   git clone https://github.com/Ansh-2002/file-uploader
   ```
2. Installation:
   ```sh
   cd client
   npm install
   npm run dev
   ```

    ```sh
   cd server
   npm install
   node server.js
   ```
